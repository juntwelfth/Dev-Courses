flexbox
    뷰포트나 요소의 크기가 불명확하거나 동적으로 변할 때에도 효율적으로 요소를 배치, 정렬, 분산할 수 있는 방법을 제공하는 새로운 레이아웃 방식
    장점을 한마디로 표현하면 '복잡한 계산 없이 요소의 크기와 순서를 유연하게 배치할 수 있다'라고 할 수 있음


수평 레이아웃
.container {
    display: flex;
}

수직 레이아웃
.container {
    display: flex;
    flex-direction: column;
}

수평 정렬
.container {
    display: flex;
    flex-direction: row;
    justify-content: center;
}

수평, 수직 정렬
.container {
    display: flex;
    flex-direction: row;
    justify-content: center;
    align-items: center;
}
